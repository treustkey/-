# Генератор Технических Заданий

Приложение для автоматизации процесса создания технических заданий в соответствии с ГОСТ 34.602-89.

## Установка

```bash
# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Linux/Mac)
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
